# The Workspace

> A cinematic 3D developer portfolio inspired by camera-driven navigation.

## 🚀 Vision

The Workspace is an interactive developer portfolio that replaces traditional scrolling with cinematic camera transitions inside a single 3D environment.

The project is inspired by the feeling of navigating a premium game menu while maintaining a professional, recruiter-friendly experience.

---

## ✨ Features

- Interactive 3D Workspace
- Camera-based Navigation
- Data-driven Architecture
- Responsive Design
- Lite Mode
- Modular Components
- Accessibility First

---

## 🛠 Tech Stack

- React 18
- TypeScript (strict mode)
- Vite + vite-ssg
- Tailwind CSS
- React Three Fiber + Drei
- GSAP (camera animations)
- Framer Motion (UI animations)
- Zustand (sliced store)
- React Router
